## Run Locally

Clone the project

```bash
  git clone https://github.com/EssaAhmd/static-page-react.git
```

Go to the project directory

```bash
  cd static-page-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  run dev
```
